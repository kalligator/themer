## Installation & usage

Install this module wherever you have `themer` installed:

    npm install @themerdev/colors-polar-ice

Then pass `@themerdev/colors-polar-ice` as a `-c` (`--colors`) arg to `themer`:

    themer -c @themerdev/colors-polar-ice ...
